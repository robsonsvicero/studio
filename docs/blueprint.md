# **App Name**: André Barbosa Imóveis

## Core Features:

- Dynamic Property Listing & Display: Showcase various properties (featured, new listings) in responsive grids, with support for images, key details, and status. Properties can be dynamically fetched for different sections of the website.
- Central Property Search: A prominent, centralized search bar allowing users to filter properties by location, type, price range, and other criteria as observed in the hero banner.
- Detailed Property View: Dedicated pages or modal views to display comprehensive information for a selected property, including high-resolution images, full descriptions, and contact options.
- Consultant 'Quem Somos' Section: An 'About Us' section dedicated to the consultant, featuring a profile picture (photo on the left as per the image), biography, and professional qualifications.
- Integrated Footer Contact Form: A contact form located in the website footer, enabling potential clients to easily submit inquiries directly from any page, requiring backend data storage.
- AI-Powered Property Inquiry Tool: A generative AI tool embedded as an assistant that can understand natural language questions from users regarding property features, neighborhood information, or general real estate advice, and provide concise, relevant responses.

## Style Guidelines:

- Main Background: Sand (#F0E6D2), providing a warm and inviting canvas.
- Text Color: Graphite (#3C3C3B), ensuring excellent readability on light backgrounds.
- Primary CTA/Accent: Moss Green (#4F5D48), used for interactive elements like buttons and calls to action, conveying nature and growth.
- Brand Detail Accent: Terracotta (#B35A44), used for subtle branding elements and highlights, adding warmth and sophistication.
- Universal Font: 'Inter' (sans-serif) for all text elements, providing a modern, clear, and highly legible experience across the site. Note: currently only Google Fonts are supported.
- Logo: 'Ativo_1.png' prominently placed in the header. General Icons: Utilize clean, vector-based line icons that align with the sophisticated aesthetic for services and property features.
- Hero Banner: Full-width hero section with a high-end interior placeholder image, featuring a centered, modern search bar for immediate property discovery.
- Property Grids: Multiple sections with responsive property cards (e.g., 'Imóveis em Destaque', 'Imóveis mais buscados'), designed to adapt seamlessly to various screen sizes.
- About Section: A two-column layout ('Quem Somos'), with the consultant's photo on the left and descriptive text content aligned on the right, providing a personal touch.
- Footer: A multi-column, organized footer including navigation links, copyright information, and an integrated contact form for user convenience.
- Subtle Hover Effects: Introduce gentle hover animations on interactive elements like property cards, buttons, and navigation links to enhance user engagement.
- Smooth Transitions: Implement fluid transitions for filtering results, page changes, and accordion-style content sections to provide a polished user experience.
